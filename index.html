Creating Automated Nutrition Control for NFT Hydroponic system

Telkom University Greenhouse Facility

This project was also being used for my Thesis. In this project I'm using TDS Sensor & Ultrasonic Sensor to sensing the condition on the nutrition liquid in the hydroponic. Then I'm using ESP8266 for collecting data from sensor and transmit the data to cloud. The cloud I was used was cloudMQTT and the protocol I was used was MQTT. Then I'm using Raspberry PI 3 as the Mini PC for collecting data from the cloud then process it using Fuzzy Logic to decide what action will the system do for controlling the nutrition of the hydroponic. The actuators of this system are connected with the Raspberry Pi, so Raspberry Pi can command the actuators directly if there are some action needed to control the nutrition in the hydroponic. Also I'm using thingspeak for the monitor dashboard. The result of this project was the system successfully to maintain the nutrition number to keep in the optimum level and help the plant to growth better than not using the system.

More details : Automated-Nutrient-Control-for-Hydroponic
